# Typescript + Rollup Boilerplate

When you want to do typescripts.

## Install

### Globals

TypeScript, Typings and TypeDoc

`npm install -g tsc typings typedoc`

### Type definitions

`typings install`

### Project

`npm install`

## Documentation

TypeDoc is located in `docs/`.
To generate TypeDoc run `npm run generate:docs`.

## Tests

`npm run test`
